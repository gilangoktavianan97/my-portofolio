Abot Me

Saya seorang Front-End Web Designer dan Junior Web Developer dengan passion menciptakan antarmuka web yang menarik, responsif, dan user-friendly. Dengan pengalaman dalam HTML, CSS, JavaScript, Bootstrap, dan Tailwind CSS, saya berfokus pada desain yang estetis sekaligus fungsional. Saya selalu mengikuti perkembangan tren desain modern dan menerapkan prinsip-prinsip UI/UX untuk memastikan pengalaman pengguna yang optimal. Selain itu, saya terus belajar framework seperti React.js dan tools desain seperti Figma untuk meningkatkan kualitas pekerjaan saya.

Sebagai seorang Junior Web Developer, saya memiliki dasar yang kuat dalam membangun website dari nol hingga deployment. Saya terbiasa bekerja dengan version control (Git/GitHub) dan mengikuti best practices dalam pengembangan web. Meskipun masih dalam tahap pembelajaran, saya antusias untuk mengembangkan keterampilan saya dalam animasi web, aksesibilitas, dan optimasi performa. Saya percaya bahwa kombinasi antara kreativitas desain dan logika pemrograman adalah kunci dalam menciptakan solusi digital yang efektif dan memukau.
